# Image and Video Processing

This repository contains the lab notebooks for an Image and Video Processing course. Each notebook covers a separate lab exercise or exam-oriented task, and the supporting image assets are stored under the `images/` directory.

## Contents

- `IVP_Lab_0.ipynb` through `IVP_Lab_8.ipynb` - individual lab notebooks
- `IVP_Lab_EXAM.ipynb` - exam practice notebook
- `images/` - reference images and sample assets used by the notebooks

## How to use

1. Open the notebook you want to work on in VS Code or Jupyter.
2. Run the cells in order unless the notebook says otherwise.
3. Keep any new output images or generated files organized under `images/` or a lab-specific subfolder.

## Notes

- The notebooks are intended to be self-contained.
- If a notebook depends on an external image, check the path inside the notebook before running it.
